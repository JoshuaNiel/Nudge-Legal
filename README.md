# Nudge — Legal Documents

This repository contains the legal documents for the Nudge iOS application.

---

## Documents

### [Privacy Policy](privacy-policy.md)

Describes what data Nudge collects, how it is used, and how it is protected. Covers:

- Account and screen time data collection
- Friend contact information and SMS consent
- How data is shared (and what it is never used for — marketing, advertising, or sale)
- Data retention periods and deletion rights
- User rights and how to contact us

### [Terms and Conditions](terms-and-conditions.md)

The terms of service for using the Nudge application. Covers:

- Description of the Nudge SMS accountability program
- SMS message frequency, rates, and carrier information
- Opt-out instructions (reply **STOP**) and support (reply **HELP**)
- Acceptable use policy and friend consent requirements
- Disclaimers, liability limits, and governing law
